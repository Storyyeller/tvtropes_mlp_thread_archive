@kegi depends on the type of script. There's the narrative script, wherin you have to set the scene and describe the characters to a certain degree, as well as the actions that the characters perform on top of their dialogue.

Dialogue script, which only contains the dialogue and character movements. It's a way of streamlining the script fo the actors.

Then there's the Camera script which comprehensively details every motion and angle that the cameras will need to be in for each scene. Dialogue is included in this version of the script because they ususally serve as cues for a camera guy to start moving
