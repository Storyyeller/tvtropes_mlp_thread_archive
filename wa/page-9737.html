#243406 Perpetual Lurker Fri, 6th Jul '12 6:50:48 PM
For as much as you bash it, Story, PHP isn't broken. Suboptimal, maybe, but the fact that 90% of the internet runs on it has to say something. The main thing it has on Python is that it's a C-based language, meaning that its simply easier to use with, port to and from, and reconcile with most other popular programming languages. I hardly even had to learn anything to use it effectively, and simple common sense and self policing can keep the nastier things that it will let you do that you shouldn't be doing from happening.

Python is great, yeah, but web applications build off of eachother, and the fact that most existing systems are in PHP means that using PHP saves time when extending, modifying, or otherwise working with them. Python needs a larger instal base before it becomes cost effective for popular use.
