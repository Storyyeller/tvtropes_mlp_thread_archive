@Applejack: That's comparing apples and oranges; they're completely different types of show. FMA is a sequential story, so it assumes familiarity with earlier episodes and caries plot points and characterization from episode to episode. MLP basically has to make every episode self contained, which requires more time spent per episode on setting up the scenes and plot. Not to mention that MLP is slice-of-life, which by definition is going to be a bit slower paced.

And yeah, Applejack has been doing really well this season; so far she's the only pony who's been in every episode.
