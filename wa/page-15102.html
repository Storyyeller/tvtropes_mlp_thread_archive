#377542 Japanese Teeth Thu, 20th Jun '13 10:01:35 PM from Meinong's jungle
@Seraphem: Ouch, that suck. Hope things start going better.

@Syd: Here's my method in GIMP 2.8. It's clunky and could probably be improved, but it works.

  1. Open image, place/resize text (black, Segoe Print). This is the easy part.

  2. Go to the "Select by Color" tool, and use it to select the text. (Don't flatten the image yet, you want the text on it's own layer)

  3. After selecting the text, go to filters > light and shadow > drop shadow

  4. This is where it gets annoying. Set both offsets at zero, make the blur radius around 7-12 (depends on the size of the text; bigger for bigger images), change the color to white, and set the opacity at 100%. I'm sure there's a way to set the defaults so you don't have to reset it all the time, but I haven't figured it out yet. This should create a white glow around the letters.

  5. Go to Select > none. Then go to Image > Flatten Image

  6. Go to File > Export and save the image.
