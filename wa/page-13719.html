Let's look at this quote from the show:

Pinkie: A Sonic Rainboom is when a pegasus like Rainbow Dash gets going so fast that they create a rainbow and a sonic boom at the same time!

Remember that Rainbow Dash, the expert, is there and doesn't contradict Pinkie. Nor does Twilight act suspicious of Pinkie's description despite her understanding of physics and magic.

Now let's break up the quote:

"a rainbow and a sonic boom at the same time"

This is confirmation that sonic booms without rainbows do happen.

"A pegasus like Rainbow Dash"

This is confirmation that other pegasi can do it.

"so fast that"

This is confirmation that the only difference between the technique in creating a sonic boom and a Sonic Rainboom is that a Sonic Rainboom is faster.

Combine this with the legendary status of the Sonic Rainboom and we learn that there are multiple pegasi who are fast enough to create a sonic boom, but that Rainbow Dash is the first in what is probably centuries to be fast enough to create a Sonic Rainboom.

This teels us that the difference in speed between a sonic boom and a Sonic Rainboom is large.

Which is all exactly what the animation tells us.
